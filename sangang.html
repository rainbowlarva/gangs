Tutorial: How to Apply Generator Output

The SanGang entry page of the MDC does not automatically format this the same way it does for arrest reports with Oggy/Monster's generator. So there's an extra step to this unless whoever maintains the MDC allows this functionality.

When you add a new SanGang entry, on the top row of text editor features you will see '</>'. This is the code display button. Press that, paste what's output by the generator and then press it again. You will then see it properly formatted as if you submitted an arrest report.

You can re-open this tutorial by pressing the question mark at the top right corner of the page.
